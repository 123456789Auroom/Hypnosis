## Прямые конкуренты (онлайн-гипнотерапевты с личным брендом)

### A. Rakitskiy — YouTube
- **Контент:** сеансы гипноза от тревоги и фобий (YouTube)
- **Позиционирование:** «Тибетское небо» — экзотический angle
- **Сильная сторона:** видео-контент, доверие через бесплатные сеансы
- **Слабая сторона:** YouTube-ориентация, не текстовый маркетинг

### Михаил Пелехатый — InstitutNLP
- **Контент:** обучение гипнозу и самогипнозу онлайн
- **Позиционирование:** НЛП-центр, образовательный подход
- **Сильная сторона:** институт = доверие, курсы как дополнительный доход
- **Слабая сторона:** не индивидуальная терапия, а обучение

### Татьяна — LotusOnlineCounselling
- **Контент:** сайт на русском, работает из Великобритании
- **Позиционирование:** русскоязычный психолог и гипнотерапевт за рубежом
- **Сильная сторона:** географическое преимущество (западный опыт)
- **Слабая сторона:** старый дизайн, мало контента

### Платформы (не прямые конкуренты, но забирают трафик)
- **Yasno.live** — самый сильный бренд (9% отбор специалистов, агрессивный маркетинг)
- **YouTalk** — психолог по Telegram (280+ специалистов)
- **Profi.ru, ProDoctorov, DocDoc** — агрегаторы

## Косвенные конкуренты

| Категория | Примеры | Угроза |
|-----------|---------|--------|
| Медитативные приложения | Insight Timer, Headspace, Calm | Низкая — не решают глубинные проблемы |
| YouTube-сеансы | Rakitskiy, бесплатные записи | Средняя — создают ощущение «я уже пробовал» |
| НЛП-коучи | Институт НЛП, Гордеев | Средняя — конкуренция за «развитие» сегмент |
| Психологи онлайн | Yasno, B17, Zigmund | Высокая — забирают «первый контакт» |
| Эзотерические практики | Таро, астрология, «энергетические чистки» | Низкая — другая аудитория |

## Конкурентные преимущества Ишвара даса

| Преимущество | Почему это важно |
|-------------|-----------------|
| **1:30 вместо 50 мин** | Глубже работа, больше результата за сессию |
| **30 000 ₽+ отсекает нецелевых** | Клиенты, которые платят — серьёзнее, мотивированнее |
| **Гарантия + бесплатный разбор** | Снимает страх «а вдруг не поможет» |
| **Тантра + мессмеризм** | Редкая комбинация, не «ещё один НЛП-коуч» |
| **Детские травмы за 1 сеанс** | Конкретный, измеримый результат |
| **Онлайн по всему миру** | Не привязан к городу |
| **Анонимность** | Не нужно показывать лицо, рассказывать друзьям |

## Позиционирование на рынке

```
Дешёвый / короткий          ← РЫНОК →          Дорогой / глубокий
─────────────────────────────────────────────────────────────
3 000–8 000 ₽                               30 000+ ₽
50 мин                                      1:30
Поверхностная релаксация                    Глубокий транс
Массовый рынок                              Премиум сегмент
         ↑                    ↑
    Средний сегмент      Ишвара даса
    (переполнен)         (свободная ниша)
```

**Стратегия:** не конкурировать с дешёвыми сессиями. Позиционироваться как премиум-работа для тех, кто уже пробовал «обычную» терапию и не получил результата.
